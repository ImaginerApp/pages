Custom Provider

Imaginer support (almost) every models available on Hugging Chat if they support Text To Image.

  1. Choose a model by searching on Hugging Face
  2. Go on the model page and click on the copy icon next to the model name
  3. Paste the name in the preferences > Custom Provider > Model
  4. Apply and enjoy!
